Appendix C. Summary of Changes

This appendix briefly describes some of the more important changes to Python since the first edition of Python Essential Reference, which described Python 1.5.2. The primary focus here is on core language changes that may cause incompatibilities between Python versions.

Detailed changes to individual library modules are not covered. For a more comprehensive list of changes, readers are encouraged to consult the documentation at http://www.python.org. The Misc/NEWS file in the Python source distribution also contains a very detailed list of language changes.
